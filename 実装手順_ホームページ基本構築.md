# ホームページ基本構築 実装手順（初心者向け）

## 📚 このガイドについて

このガイドは、サッカー教室ホームページの**基本的な見た目と構造**を作成するための手順書です。データベース設計の前に、まずホームページの骨組みを作成します。

**所要時間**: 約2〜3時間（初めての場合）

---

## 📖 用語集（最初に読んでおくと理解が深まります）

### Next.jsに関する用語

- **コンポーネント**: 再利用可能なUIの部品。例: ボタン、ヘッダー、カードなど
- **ページ**: ブラウザで表示される画面。例: トップページ、お問い合わせページ
- **レイアウト**: 複数のページで共通して使う構造（ヘッダー、フッターなど）
- **App Router**: Next.jsの新しいルーティング方式。`app`フォルダ内にファイルを置くと自動的にページになります
- **Tailwind CSS**: ユーティリティファーストのCSSフレームワーク。クラス名でスタイルを指定します

```typescript
              onClick={() => setIsMenuOpen(false)}
            >

              href="/about" 
              className="block py-2 text-gray-700 hover:text-green-600"
              onClick={() => setIsMenuOpen(false)}

            >

      </nav>
    </header>
  )
}
```

**コードの説明**:
- `'use client'`: クライアントコンポーネント（状態管理が必要なため）
- `useState`: モバイルメニューの開閉状態を管理
- `Link`: Next.jsのリンクコンポーネント（ページ遷移が高速）
- `md:hidden` / `md:flex`: Tailwind CSSのレスポンシブクラス（モバイル/デスクトップで表示を切り替え）

### 2-4. Buttonコンポーネントの作成（オプション）

**目的**: 再利用可能なボタンコンポーネントを作成します。

**手順**:

1. **`frontend/components/Button.tsx`ファイルを作成**

2. **以下のコードをコピー＆ペースト**

```typescript
import { ReactNode } from 'react'

interface ButtonProps {
  children: ReactNode
  onClick?: () => void
  href?: string
  variant?: 'primary' | 'secondary' | 'outline'
  className?: string
}

export default function Button({ 
  children, 
  onClick, 
  href, 
  variant = 'primary',
  className = '' 
}: ButtonProps) {
  const baseClasses = 'px-6 py-3 rounded-lg font-semibold transition duration-200'
  
  const variantClasses = {
    primary: 'bg-green-600 text-white hover:bg-green-700',
    secondary: 'bg-gray-600 text-white hover:bg-gray-700',
    outline: 'border-2 border-green-600 text-green-600 hover:bg-green-50'
  }

  const classes = `${baseClasses} ${variantClasses[variant]} ${className}`

  if (href) {
    return (
      <a href={href} className={classes}>
        {children}
      </a>
    )
  }

  return (
    <button onClick={onClick} className={classes}>
      {children}
    </button>
  )
}
```

2. **`frontend/app/contact/page.tsx`ファイルを作成**

3. **以下のコードをコピー＆ペースト**

**⚠️ 重要**: このファイルでは`useState`を使うため、**必ずファイルの先頭に`'use client'`を書いてください**。これを忘れるとエラーが発生します。

```typescript
'use client'

import { useState } from 'react'

export default function Contact() {
  const [formData, setFormData] = useState({
    name: '',
    email: '',
    message: ''
  })

  const handleSubmit = (e: React.FormEvent) => {
    e.preventDefault()
    // 後で実装: フォーム送信処理
    alert('お問い合わせありがとうございます。後ほどご連絡いたします。')
    setFormData({ name: '', email: '', message: '' })
  }

  return (
    <div className="container mx-auto px-4 py-16">
      <h1 className="text-4xl font-bold text-center mb-12 text-gray-900">
        お問い合わせ
      </h1>

      <div className="max-w-2xl mx-auto">
        {/* お問い合わせフォーム */}
        <form onSubmit={handleSubmit} className="bg-white p-8 rounded-lg shadow-md space-y-6">
          <div>
            <label htmlFor="name" className="block text-sm font-medium text-gray-700 mb-2">
              お名前 <span className="text-red-500">*</span>
            </label>
            <input
              type="text"
              id="name"
              required
              value={formData.name}
              onChange={(e) => setFormData({ ...formData, name: e.target.value })}
              className="w-full px-4 py-2 border border-gray-300 rounded-lg focus:ring-2 focus:ring-green-500 focus:border-transparent"
            />
          </div>

          <div>
            <label htmlFor="email" className="block text-sm font-medium text-gray-700 mb-2">
              メールアドレス <span className="text-red-500">*</span>
            </label>
            <input
              type="email"
              id="email"
              required
              value={formData.email}
              onChange={(e) => setFormData({ ...formData, email: e.target.value })}
              className="w-full px-4 py-2 border border-gray-300 rounded-lg focus:ring-2 focus:ring-green-500 focus:border-transparent"
            />
          </div>

          <div>
            <label htmlFor="message" className="block text-sm font-medium text-gray-700 mb-2">
              お問い合わせ内容 <span className="text-red-500">*</span>
            </label>
            <textarea
              id="message"
              required
              rows={6}
              value={formData.message}
              onChange={(e) => setFormData({ ...formData, message: e.target.value })}
              className="w-full px-4 py-2 border border-gray-300 rounded-lg focus:ring-2 focus:ring-green-500 focus:border-transparent"
            />
          </div>

            type="submit"
            className="w-full bg-green-600 text-white py-3 rounded-lg font-semibold hover:bg-green-700 transition"
          >
            送信
          </button>
        </form>
```

## 🎨 カスタマイズのヒント

### 色を変更したい場合

Tailwind CSSの色クラスを変更します：
- `green-600` → `blue-600`（青色に変更）
- `green-600` → `red-600`（赤色に変更）

### フォントを変更したい場合

`layout.tsx`の`Inter`を別のフォントに変更できます：
```typescript
import { Noto_Sans_JP } from "next/font/google";

const notoSansJP = Noto_Sans_JP({ subsets: ["latin"] });
```

### ロゴ画像を追加したい場合

1. `public`フォルダに画像を配置（例: `public/logo.png`）
2. Headerコンポーネントで画像を使用：
```typescript
import Image from 'next/image'

<Image src="/logo.png" alt="ロゴ" width={100} height={50} />
```

## ⚠️ トラブルシューティング

### エラー1: useStateを使うとエラーが出る

**エラーメッセージ例**:
```
You're importing a component that needs `useState`. This React Hook only works in a Client Component. To fix, mark the file (or its parent) with the `"use client"` directive.
```

**原因**: `useState`などのReact Hooks（フック）を使うには、ファイルの先頭に`'use client'`を書く必要があります。

**なぜ必要？**: 
- Next.jsのApp Routerでは、デフォルトで**サーバーコンポーネント**として動作します
- サーバーコンポーネントでは、`useState`や`onClick`などの**インタラクティブな機能**が使えません
- `'use client'`を書くことで、**クライアントコンポーネント**として動作し、これらの機能が使えるようになります

**解決方法**:

1. **ファイルの先頭に`'use client'`を追加**
   - `frontend/app/contact/page.tsx`ファイルを開く
   - ファイルの**一番最初の行**に`'use client'`を追加

   **修正前**:
   ```typescript
   import { useState } from 'react'
   ```

   **修正後**:
   ```typescript
   'use client'

   import { useState } from 'react'
   ```

2. **保存して確認**
   - ファイルを保存（`Ctrl + S`）
   - ブラウザをリロードしてエラーが解消されたか確認

**どのファイルに必要？**:
- `useState`、`useEffect`などのReact Hooksを使うファイル
- `onClick`、`onChange`などのイベントハンドラーを使うファイル
- 例: `Header.tsx`、`Contact`ページなど

**どのファイルに不要？**:
- 静的なコンテンツだけを表示するファイル
- 例: `About`ページ、`Footer.tsx`など

---

### エラー2: コンポーネントが見つからない

**エラーメッセージ例**:
```
Module not found: Can't resolve '@/components/Header'
```

**解決方法**:
- `components`フォルダが`frontend`フォルダの直下にあるか確認
- ファイル名が`Header.tsx`（大文字のH）になっているか確認

---

### エラー3: ページが表示されない

**解決方法**:
- 開発サーバーが起動しているか確認（`npm run dev`）
- ブラウザのコンソールでエラーを確認（F12キー）
- ファイルの保存を確認（`Ctrl + S`）

---

### エラー4: スタイルが適用されない

**解決方法**:
- Tailwind CSSのクラス名が正しいか確認
- `globals.css`にTailwind CSSのディレクティブが含まれているか確認：
```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

## 💡 よくある質問（FAQ）

### Q1: 新しいページを追加したい場合は？

**A**: `app`フォルダ内に新しいフォルダを作成し、その中に`page.tsx`を配置します。
例: `app/news/page.tsx` → `http://localhost:3000/news`

---

### Q2: コンポーネントを再利用するには？

**A**: `components`フォルダにコンポーネントを作成し、他のファイルでインポートします：
```typescript
import Header from '@/components/Header'
```

---

### Q3: 画像を追加するには？

**A**: `public`フォルダに画像を配置し、以下のように使用します：
```typescript
<Image src="/画像名.png" alt="説明" width={100} height={100} />
```
